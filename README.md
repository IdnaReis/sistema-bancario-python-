## 📋 Descrição
Sistema bancário simples desenvolvido em Python como parte do desafio da plataforma DIO. O sistema permite realizar operações básicas de uma conta bancária.

## ⚙️ Funcionalidades
- 💰 *Depósito* — Permite depositar valores positivos na conta
- 💸 *Saque* — Limite de 3 saques diários com valor máximo de R$ 500,00 por saque
- 📄 *Extrato* — Lista todos os depósitos e saques realizados com saldo atual
